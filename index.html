Acesso Local

Com o servidor Node (npm start): informe a senha igual a ADMIN_LOCAL_PASS no .env (pasta do projeto, junto ao server.js), na mesma URL/porta (ex.: http://localhost:3000/). Senha com @: use aspas no .env, ex. ADMIN_LOCAL_PASS="AGENDA@2026".

Se a API responder que não há senha local (ex.: Vercel sem ADMIN_LOCAL_PASS), o portal abre direto o login Supabase. Para nunca pedir este passo: PORTAL_SKIP_LOCAL_ADMIN_GATE=true ou skipLocalAdminGate: true no portal-config.js.
